# INSTALL

Requirements:
Node.js 18 or newer

Clone project:
```
git clone https://github.com/yourname/thenull-v2ray-toolkit.git
cd thenull-v2ray-toolkit
```

Install dependencies:
```
npm install
```
Run server:
```
node server.js
```
Open in browser:

http://localhost:3000

# README (English)


## About

TheNull V2ray Tool Kit is a good web-based toolkit for managing, converting and testing
V2Ray / Xray configurations.

This project is designed for real-world usage and real TCP network testing.

---

## Features (v1.0.0)

- Batch rename for:
  - VLESS
  - VMess
  - Trojan
  - Shadowsocks

- Automatic numbering when multiple configs are renamed

- JSON to link converter:
  - VLESS
  - VMess
  - Trojan

- Real TCP latency test (server side)

- Copy button for all outputs

- Basic multilingual interface

- Light and dark theme

- Separate panel for NPVT file inspection

## معرفی پروژه

TheNull V2ray Tool Kit یک ابزار تحت وب خوب برای مدیریت، تبدیل و تست
