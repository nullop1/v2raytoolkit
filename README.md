## About

TheNull V2ray Tool Kit is a modern web-based toolkit for managing, converting and testing
V2Ray / Xray configurations.

This project is designed for real-world usage and real TCP network testing.

---

## Features (v1.0.0)

- Batch rename for:
  - VLESS
  - VMess
  - Trojan
  - Shadowsocks

- Automatic numbering when multiple configs are renamed

- JSON to link converter:
  - VLESS
  - VMess
  - Trojan

- Real TCP latency test (server side)

- Copy button for all outputs

- Basic multilingual interface

- Light and dark theme

- Separate panel for NPVT file inspection

---

## Limitations

This project cannot decrypt or unlock:

- NetMod locked subscriptions
- NPV Tunnel locked exports

These formats use application-level encryption.

## معرفی پروژه

TheNull V2ray Tool Kit یک ابزار تحت وب مدرن برای مدیریت، تبدیل و تست
کانفیگ‌های V2Ray و Xray می‌باشد.

این پروژه برای استفاده واقعی و تست واقعی شبکه طراحی شده است.

---

## امکانات نسخه 1.0.0

- تغییر نام گروهی کانفیگ‌های:
  - VLESS
  - VMess
  - Trojan
  - Shadowsocks

- شماره‌گذاری خودکار هنگام تغییر نام چند کانفیگ

- تبدیل JSON به لینک:
  - VLESS
  - VMess
  - Trojan

- تست پینگ واقعی TCP از سمت سرور

- دکمه کپی برای تمام خروجی‌ها

- رابط کاربری چندزبانه پایه

- حالت شب و روز

- بخش جداگانه برای بررسی فایل NPVT

---

## محدودیت‌ها

این ابزار قادر به دیکریپت یا باز کردن موارد زیر نیست:

- اشتراک‌های lock شده NetMod
- خروجی‌های lock شده NPV Tunnel

این فرمت‌ها از رمزنگاری داخلی برنامه استفاده می‌کنند.
